getMergeCol

Description

  • ● 입력한 인덱스 번호에 해당하는 컬럼의 merge 설정 여부를 반환합니다.

Parameter

	
	 * 메소드 : datagrid.getMergeCol(param1);
	   - param1 (필수) : [number] merge 설정 여부를 반환할 컬럼 인덱스
