getColList

Description

  • ●특정 ref | caption에 해당하는 열 인덱스를 반환합니다.

Parameter

	
	 * 메소드 : datagrid.getColList(param1, param2);
	   - param1 (필수) : [string] 검색 조건
		  → 'ref'		: ref로 검색
		  → 'caption'	: caption으로 검색
	   - param2 (필수) : [string] 검색 조건 값
