setMergeCells

Description

  • ● 그리드의 데이터 영역을 입력한 병합 타입으로 병합합니다.

Parameter

	
	 * 메소드 : datagrid.setMergeCells(param1);
	   - param1 (필수) : [string] 병합 설정 여부
		  → 'none'		: 병합 없음
		  → 'bycol'		: 동일한 데이터 값을 갖는 이웃한 셀들을 열 단위로 병합
	          → 'byrow'		: 동일한 데이터 값을 갖는 이웃한 셀들을 행 단위로 병합
	          → 'bycolrec'		: 동일한 데이터 값을 갖는 경우 열단위 병합후, 수행이후 병합된 결과에 따라 행단위로 병합
		  → 'byrestriccol'	: 첫번째 열부터 순서대로 행병합 후 병합 속성을 적용한 바로 앞 열의 병합 결과에 따라 병합
