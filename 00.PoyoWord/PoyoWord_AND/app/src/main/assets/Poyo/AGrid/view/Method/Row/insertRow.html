insertRow

Description

  • ● 지정한 행 인덱스의 위/아래에 행을 삽입합니다.

Parameter

	
	 * 메소드 : datagrid.insertRow(param1, param2, param3, param4);
	   - param1 (필수) : [number] 기준 행 인덱스
	   - param2 (필수) : [string] 기준 행 기준 행 추가 위치
	   	  → 'below' (default)	: 기준 행 아래에 행 추가
		  → 'above'			: 기준 행 위에 행 추가
	   - param3 (선택) : [string | array | json object]  입력할 데이터
   	   - param4 (선택) : [boolean]  행 추가 후 스크롤 이동 여부
		  → true (default)		: 스크롤 이동
		  → false				: 스크롤 이동하지 않음
