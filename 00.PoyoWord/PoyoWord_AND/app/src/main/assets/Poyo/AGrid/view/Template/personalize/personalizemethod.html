personalize (method)

Description

  • ● 그리드의 너비, 컬럼 위치, 숨김 등 그리드의 개인화를 설정할 수 있습니다.
  • ● 주의사항 : ie의 경우는 server에 올려서 테스트 하셔야 합니다. chrome은 html 파일로도 테스트 및 확인이 가능합니다.

Value


	 * 메소드 : datagrid.savePersonalInfo(param1);
	   - param1 (필수) : [string]  개인화를 저장할 ID
	   
	 * 메소드 : datagrid.loadPersonalInfo(param1);
	   - param1 (필수) : [string]  개인화를 호출할 ID

	function fnSave(){
