format

Description

  • ● 컬럼의 포맷을 설정합니다.
  • ● 실제 데이터는 format의 적용을 받아 변경되지 않고, 화면에 보이는 부분만 적용됩니다.

Value

			
	 * 컬럼속성 : format : {type: value1, rule: value2, origin: value3, locale : value4, usetime : value5, callback: value6}
	   - value1 (필수) : [string] 표현하려는 데이터 타입
		  → 'date'					: 날짜 또는 시간 형식
		  → 'number'					: 자릿수 구분 또는 소수점 표시 형식
		  → 'string'					: 특정 문자 형식 
		  → 'custom'					: 사용자 지정 형식
	   - value2 (필수) : [string] 표현하려는 데이터 형태
		  → (date) 'yyyy' | 'yy'			: 4자리 또는 2자리 년 표시
		  → (date) 'mmm' | 'mm' | 'm'	: '월' 출력 또는 2자리 또는 1자리 월 표시
		  → (date) 'do' | 'dd' | 'd'		: 일' 출력 또는 2자리 또는 1자리 일 표시
		  → (date) 'HH' | 'hh'			: 24시간 또는 12시간 표시
		  → (date) 'mm' | 'm'			: 2자리 또는 1자리 분 표시
		  → (date) 'ss' | 's'				: 2자리 또는 1자리 초 표시
		  → (number) '000.0'			: 빈자리를 0으로 표시, 소수점 둘째자리 이하 반올림
		  → (number) '#,###.##'		: 3자리마다 '.' 표시, 유효한 자리만 표시, 소수점 셋째자리 이하 반올림
		  → (string) '@ "님"'			: @ 문자 표시
		  → (string) '"양수";"음수"'		: 데이터 값에 따라 양수와 음수로 조건부 표시. 0 미만인 숫자만 음수로 표시
		  → (string) '# ?/?'				: 실수를 분수 형태로 표시
		  → (string) '#,### "원"'			: 숫자 자리수 구분 및 글자 동시 표시	
	   - value3 (필수) : [string] date 포맷 적용시 원본 데이터 형식
		  → (date) 'yyyy' | 'yy'			: 4자리 또는 2자리 년 표시
		  → (date) 'mmm' | 'mm' | 'm'	: '월' 출력 또는 2자리 또는 1자리 월 표시
		  → (date) 'do' | 'dd' | 'd'		: 일' 출력 또는 2자리 또는 1자리 일 표시
		  → (date) 'HH' | 'hh'			: 24시간 또는 12시간 표시
		  → (date) 'mm' | 'm'			: 2자리 또는 1자리 분 표시
		  → (date) 'ss' | 's'				: 2자리 또는 1자리 초 표시
	   - value4 (선택) : [string] date 및 number 포맷 적용시 locale 에 따라 Default Format 및 표시 상태가 달라지도록 선언
		  → (date, number) 'ko' (default)	: 한국 표시 형식
		  → (date, number) 'us'			: 미국 표시 형식
	   - value5 (선택) : [boolean] date 포맷 형태에 시분초 표시 여부
		  → (date) true (default)			: 시분초 표시 (날짜 시간 동시에 표시할 경우 rule에 날짜 부분은 대문자로 선언)
		  → (date) false				: 시분초 표시하지 않음
	   - value6 (선택) : [string | object] custom 포맷 형태 사용시 호출하는 사용자가 원하는 형태의 function
		  → (custom) param1 (선택)		: 실제 데이터
Example 
			
	var datagrid;
	var SBGridProperties = {};
	
	function createGrid(){
	
		SBGridProperties.parentid = 'SBGridArea';
		SBGridProperties.id = 'datagrid';
		SBGridProperties.jsonref = 'formatData';
		SBGridProperties.selectmode = 'free';
		SBGridProperties.columns = [
			{caption : ['기본정보','이름'],		ref : 'name',	width : '80px',	style : 'text-align:center',	type : 'input',		format : {type:'string', rule:'@ "사원"'}},
			{caption : ['기본정보','생년월일'],	ref : 'birth',	width : '90px',	style : 'text-align:center',	type : 'datepicker',	format : {type:'date', rule:'yyyy-mm-dd', origin : 'yyyymmdd',locale:'ko' },	typeinfo : {locale : 'ko' , dateformat :'yymmdd'} },
			{caption : ['부수정보','기록'],		ref : 'time',	width : '70px',	style : 'text-align:center',	type : 'output',	format : {type:'date', rule:'HH:mm:ss', origin : 'HHmmss' } },
			{caption : ['부수정보','점수'],		ref : 'score',	width : '70px',	style : 'text-align:center',	type : 'output',	format : {type:'number', rule:'#,###.##'} },
			{caption : ['부수정보','주소'],		ref : 'addr',	width : '250px',	style : 'text-align:center',	type : 'textarea',	format : {type:'custom', callback : fnCustomFormat}  }
		];
		datagrid = _SBGrid.create(SBGridProperties);
	};
	function fnCustomFormat(strValue) {
		return "대한민국 "+strValue;
	}
