setDnD

Description

  • ● 두 그리드 간의 Drag & Drop 을 통한 행 데이터 이동이 가능합니다.

Value

	
	 * 메소드 : datagrid1.setDnD({target : value1, type : value2, position : value3, stylesync : value4});
	   - value1 : [string] 데이터를 drop할 그리드 객체 (그리드 id)
	   - value2 : [string] 데이터 이동 유형
		  → 'copy' (default)		: drag되는 그리드에서 데이터 유지
		  → 'move'			: drag되는 그리드에서 데이터 삭제
	   - value3 : [string] 데이터 이동 위치
		  → 'insert' (default)	: 모든 페이지 데이터 정렬
		  → 'add'				: 한 페이지 내에서 데이터 정렬
	   - value4 : [boolean] type이 move일 때 데이터 drag시 스타일 이동 여부
		  → true				: 스타일 유지
		  → false	(default)		: 스타일 이동
