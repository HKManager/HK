checkbox

Description

  • ● 컬럼 type 중 checkbox type을 설정합니다.
  • ● 체크박스 형태의 선택 유형입니다.

Value

			
	 * typeinfo : {checkedvalue : value1, uncheckedvalue : value2}
	   - value1 (선택) : [string] checkbox 선택시의 데이터 값 설정 (default : 'true')
	   - value2 (선택) : [string] checkbox 해제시의 데이터 값 설정 (default : 'false')
