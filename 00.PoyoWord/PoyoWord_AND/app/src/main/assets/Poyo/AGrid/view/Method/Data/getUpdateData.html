getUpdateData

Description

  • ● 그리드 전체에서 상태정보(추가/갱신/삭제)가 추가된 행만 JSON 타입(객체 or 문자열)으로 반환합니다.

Parameter

	
	 * 메소드 : datagrid.getUpdateData(param1, param2, param3, param4);
	   - param1 (선택) : [boolean] 상태정보 데이터를 JSON(객체) 형태로 반환 여부
		  → true			: json object 형식으로 반환
		  → false (default)	: json string 형식으로 반환
	   - param2 (선택) : [string] 상태정보 데이터 중 반환하려는 종류 설정 
		  → 'all' (default)	: 모든 상태 정보를 가진 데이터 반환
		  → 'i'			: insert 상태 정보를 가진 데이터 반환
		  → 'd'			: delete 상태 정보를 가진 데이터 반환
		  → 'u'			: update 상태 정보를 가진 데이터 반환
	   - param3 (선택) : [string] 상태정보 데이터에서 상태정보 속성의 이름(default : 'status')
	   - param4 (선택) : [string] 상태정보 데이터에서 행 인덱스 속성의 이름(default : 'rownum')
