getFixedcellcheckboxChecked

Description

  • ● 특정 셀에 열 전체 체크박스가 있는 경우 해당 체크 여부를 반환합니다.

Parameter

			
	* 메소드 : datagrid.getFixedcellcheckboxChecked(param1,param2);
	    - param1 (필수)  : [number] 전체 체크박스 행 인덱스
	    - param2 (필수)  : [number] 전체 체크박스 열 인덱스
