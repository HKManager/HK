SBGrid 2.5 테마 구성

  • ● 기존 테마에 Navy, Gray 테마가 추가되었습니다.
  • ● Default : SBGrid.css, SBGrid_Default.css
  • ● Navy : SBGrid_Navy.css, SBGrid_Navy_Default.css
  • ● Gray : SBGrid_Gray.css, SBGrid_Gray_Default.css
  • ※ SBGrid2.5.1543 버전부터 스타일 테마를 적용하실 수 있습니다.

SBGrid 2.5 테마 적용 방법

  • 1. 적용할 테마에 맞는 변수를 SBpath 아래에 선언합니다.
    • Ex) SBtheme = “gray”;
    • ※ 테마 스타일 적용 시 SBtheme 는 반드시 소문자로 기재해 주셔야 합니다.
  • 2. Default : SBGrid.css, SBGrid_Default.css
    • Ex)
    • <link href="../SBGrid/css/SBGrid_Gray.css" rel="stylesheet" type="text/css">
    • <link href="../SBGrid/css/SBGrid_Gray_Default.css" rel="stylesheet" type="text/css">

Example

			
    <!DOCTYPE html>
    <html>
        <head>
            <title>Style Sample</title>

            <META http-equiv="Content-Type" content="text/html; charset=UTF-8">
            <meta content="initial-scale=1.0" name="viewport">
            <meta content="IE=edge, chrome=1" http-equiv="X-UA-Compatible">

            <script language="javascript">
                var SBpath = "./";
                var SBtheme = "gray";
            <script>

            <!-- SBGrid CSS -->
            <link href="./SBGrid/css/SBGrid_Gray.css" rel="stylesheet" type="text/css">      // SBGrid v2.5 Library인 'SBGrid'폴더 상대경로
            <link href="./SBGrid/css/SBGrid_Gray_Default.css" rel="stylesheet" type="text/css">  // CSS 스타일 적용 시 필수항목 입니다.
