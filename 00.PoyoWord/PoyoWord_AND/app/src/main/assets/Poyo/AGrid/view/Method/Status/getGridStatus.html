getGridStatus

Description

  • ● 그리드의 상태정보(추가/갱신/삭제)를 특정 형식의 문자열으로 반환합니다.

Parameter

	
	 * 메소드 : datagrid.getGridStatus(param1, param2, param3);
	   - param1 (선택) : [string] 반환하는 상태 정보의 종류
		  → 'all' (default)	: 모든 상태 정보
		  → 'i' | 'insert'		: insert 상태 정보
		  → 'u' | 'update'	: update 상태 정보
		  → 'd' | 'delete' 	: delete 상태 정보
	   - param2 (선택) : [string] 반환하는 상태 정보의 속성명 (default : 'm')
	   - param3 (선택) : [string] 반환하는 행 인덱스 속성명 (default : 'rownum')
