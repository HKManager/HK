selectRows

Description

  • ● 지정한 행들에 파란 배경색으로 하이라이트를 표시합니다.

Parameter

	
	 * 메소드 : datagrid.selectRows(param1);
	   - param1 (필수) : [number array] 행 인덱스 배열
