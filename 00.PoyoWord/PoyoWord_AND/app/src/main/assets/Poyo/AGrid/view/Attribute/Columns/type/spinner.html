spinner

Description

  • ● 컬럼 type 중 spinner type을 설정합니다.
  • ● 위/아래 버튼으로 숫자 값 입력이 가능한 유형입니다.

Value

			
	 * typeinfo : {max : value1, min : value2, step : value3,  oneclickedit : value4, mask : {alias: value5, unmaskvalue : value6}}
            - value1 (선택) : [number] 위/아래 버튼으로 입력 가능한 최대 값
            - value2 (선택) : [number] 위/아래 버튼으로 입력 가능한 최소 값
            - value3 (선택) : [number] 위/아래 버튼 클릭시 설정 값 만큼 숫자 증가 및 감소
            - value4 (선택) : [boolean] 마우스 한 번의 클릭으로 편집 모드 활성화 여부
              → true			: 마우스 한 번 클릭으로 편집
              → false (default)	: 마우스 더블 클릭으로 편집
            - value5 (선택) : [number] 편집 모드에서 데이터 표시 형식
            - value6 (선택) : [boolean] 실제 데이터를 alias를 적용하지 않은 입력 데이터로 변경 여부
              → true (default)	: alias를 적용하지 않은 실제 입력한 데이터로 변경
              → false		: alias를 적용한 데이터로 변경
