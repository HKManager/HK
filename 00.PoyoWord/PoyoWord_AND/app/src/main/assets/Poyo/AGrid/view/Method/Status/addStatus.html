addStatus

Description

  • ● 지정한 행의 상태정보(추가/갱신/삭제)를 중복 실행하여 중첩된 상태값을 설정합니다.
  • ● 상태정보를 + 하는 개념입니다. (insert+update, insert+delete, update+delete, insert+update+delete)

Parameter

	
	 * 메소드 : datagrid.addStatus(param1, param2);
	   - param1 (필수) : [number] 행 인덱스
	   - param2 (필수) : [number | string] 추가할 상태값
		  → 0 | 'new'		: new 상태 추가
		  → 1 | 'insert'		: insert 상태 추가
		  → 2 | 'update'	: update 상태 추가
		  → 4 | 'delete'	: delete 상태 추가
