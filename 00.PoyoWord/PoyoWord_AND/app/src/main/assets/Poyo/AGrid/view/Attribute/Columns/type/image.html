image

Description

  • ● 컬럼 type 중 image type을 설정합니다.
  • ● 이미지를 그리드에 표시하고 이벤트를 호출할 수 있는 출력 유형입니다.

Value

			
	 * typeinfo : {imageuri : value1, imagewidth : value2, imageheight : value3, imagestyle : value4, imagealt : value5, imageclick : value6, imageerroruri : value7 }
	   - value1 (필수) : [string] image 경로
	   - value2 (선택) : [string] image 너비
	   - value3 (선택) : [string] image 높이
	   - value4 (선택) : [string] image 스타일
	   - value5 (선택) : [string] image에 커서를 올렸을 때의 alt
	   - value6 (선택) : [string] image 클릭시 호출하는 function
	   - value7 (선택) : [string] 에러 image 경로
