setCellDisabled

Description

  • ● 지정한 셀의 disabled 상태를 설정합니다.

Parameter

	
	 * 메소드 : datagrid.setCellDisabled(param1, param2, param3, param4, param5, param6);
	   - param1 (필수) : [number] 시작 행 인덱스
	   - param2 (필수) : [number] 시작 열 인덱스
	   - param3 (필수) : [number] 종료 행 인덱스
	   - param4 (필수) : [number] 종료 열 인덱스
	   - param5 (필수) : [boolean] disabled 여부
		  → true			: disabled 설정
		  → false			: disabled 해제 
	   - param6 (선택) : [boolean] disabled 상태 변경 후 그리드의 refresh 여부
		  → true			: 메소드 작동 후 바로 refresh
		  → false (default)	: 메소드 작동 후 refresh 하지않음

	function setCellDisabled1(){
		var nRow = datagrid.getRow();
		var nCol = datagrid.getCol();
		datagrid.setCellDisabled(nRow, nCol, nRow, nCol, true, false);
