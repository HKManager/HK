getColRef

Description

  • ● 특정 ref명을 매핑하는 열 인덱스를 반환합니다.
  • ● ref명을 여러 열에서 매핑하는 경우 첫번째 열 인덱스를 반환합니다.

Parameter

	
	 * 메소드 : datagrid.getColRef(param1);
	   - param1 (필수) : [string] ref명
