addRow

Description

  • ● 그리드 마지막 행 다음 위치에 행을 추가합니다.

Parameter

	
	 * 메소드 : datagrid.addRow(pamra1,param2);
	   - param1 (선택) : [boolean] 행추가 후 그리드 refresh 여부
		  → true (default)	: 메소드 작동 후 바로 refresh 
		  → false			: 메소드 작동 후 refresh 하지 않음
	   - param2 (선택) : [string | array | json object]  입력할 데이터
