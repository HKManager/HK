mergecellsfixedrows

Description

  • ● 그리드내에서 헤더 영역의 병합 타입을 설정합니다.

Value

	
	 * 속   성 : SBGridProperties.mergecellsfixedrows = value1;
	   - value1 : [string] 병합 타입
	          → 'bycolrec' (default)	: 상하로 이웃한 헤더의 값이 동일할 경우 상하로 병합후, 병합된 결과에 따라 좌우로 병합
		  → 'bycol'			: 상하로 이웃한 헤더의 값이 동일할 경우 상하로 병합
		  → 'byrow'			: 좌우로 이웃한 헤더의 값이 동일할 경우 좌우로 병합
		  → 'byrestriccol'		: 첫번째 열부터 동일한 값의 헤더를 순서대로 상하병합한 후 다음열의 병합은 앞 열의 병합 결과에 따라 병합
