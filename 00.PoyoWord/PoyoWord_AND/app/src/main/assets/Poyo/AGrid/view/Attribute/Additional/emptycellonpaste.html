emptycellonpaste

Description

  • ● date format에 맞지 않는 데이터를 빈값으로 붙여넣을지 여부를 설정합니다.

Value


	 * 속   성 : SBGridProperties.emptycellonpaste = value1;
	   - value1 : [boolean] date format에 맞지 않는 데이터 붙여넣기시 공백 여부
		  → true (default) 	: date format이 맞지 않을 때 붙여넣기시 공백 처리
		  → false 			: date format이 맞지 않을 때 붙여넣기시 원본 데이터로 보임
Example
