setColStyles

Description

  • ● 지정한 여러 열에 스타일을 설정합니다.
  • ● 복수의 열에 여러 CSS 구문을 담을 수 있습니다.
  • ● 주의사항 : ie의 경우, css 구문 내부에 띄어쓰기를 할 경우 스타일이 지정되지 않을 수 있습니다.

Parameter

			
	 * 메소드 : datagrid.setColStyles(param1, param2, param3);
	    - param1 (필수) : [number | number array] 열 인덱스 (배열 형태로 [ ] 복수 열 인덱스 가능)
	    - param2 (필수) : [string] 스타일 구문
	    - param3 (필수) : [string] 스타일 적용 범위
		  → 'fixed'		: header 영역 
		  → 'data'		: data 영역	 
		  → 'all'			: 전체 영역

	function evnClick(){
