setComboFiltering

Description

  • ● 입력한 인덱스번호에 해당하는 컬럼에 콤보 메뉴를 입력받은 필터링리스트에 값으로 적용합니다.

Parameter


	 * 메소드 : datagrid.setComboFiltering(param1,[{attrname : value1, attrvalue : value2}]);
	 - param1 (필수) : [number] 열 인덱스
	 - value1 (필수) : [string] 필터링하려는 데이터의 속성명
	 - value2 (필수) : [string] 필터링하려는 데이터 값
