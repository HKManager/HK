rowdatatype

Description

  • ● 그리드의 데이터 형태를 설정합니다.
  • ● 대용량 데이터 처리를 위해서 통신 트래픽을 줄이기 위해 2차원 배열 형식으로 그리드에 매핑할 수 있습니다.
  • ● columns 에 입력되는 ref 값을 0, 1, 2, .. 의 Number 형태로 설정합니다.

Value

	
	 * 속   성 : SBGridProperties.rowdatatype = value1;
	   - value1 : [string] 그리드에 매핑할 json 데이터의 형태
	          → 'object' (default)	: key와 value로 구성된 json object array 형태
	          → 'array'			: value로만 구성된 array 형태
