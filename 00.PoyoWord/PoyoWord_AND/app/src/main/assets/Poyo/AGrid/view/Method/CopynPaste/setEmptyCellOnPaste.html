setEmptyCellOnPaste

Description

  • ● date format 셀에 데이터 붙여넣기시 format에 맞지 않는 데이터를 빈값으로 표시할지 여부를 설정합니다.
  • ● 붙여넣기가 가능(SBGridProperties.allowpaste = true)할 경우 사용가능

Parameter


	  * 메소드 : datagrid.setEmptyCellOnPaste(param1);
	  - param1 (필수) : [boolean] format에 맞지 않는 데이터를 빈값으로 표시할지 여부
		  → true 		: date format이 맞지 않을 때 붙여넣기시 공백 처리
		  → false 		: date format이 맞지 않을 때 붙여넣기시 원본 데이터로 보임
