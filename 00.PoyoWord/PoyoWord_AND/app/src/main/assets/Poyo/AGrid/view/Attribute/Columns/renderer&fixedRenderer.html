renderer & fixedRenderer

Description

  • ● 셀 데이터의 정보를 이용하여 html 태그를 삽입할 수 있도록 설정합니다.
  • ● 데이터 영역과 헤더 영역에서 각각 설정할 수 있습니다.
  • ● 주의 사항
  • - output type 컬럼에서만 이용할 수 있습니다.
  • - 내부 html소스의 id, name이 중복 되면 문제가 발생 할 수 있습니다
  • - 렌더러 스타일에 width, height를 주실 때 그리드 row의 height와 column의 width보다 작아야 합니다.
  • - CSS 충돌이 일어날 수 있기 때문에 주의가 필요합니다.(그리드가 Table 구조를 이루고 있기 때문 입니다.)
  • - 렌더러 기능 사용시 getUpdateData, getGridStatus, getCellData, getRowData등의 그리드 메소드로 값을 불러올 수 없기 때문에 jquery 등의 그리드 외적인 방식으로 값을 불러와야 합니다.

Value

			
	 * 컬럼속성 : renderer : value1 | fixedRenderer : value1
	   - value1 (필수) : [object] function 지정 or function 호출
		  → param1 (선택)  : [object] 선택한 그리드 객체(id)
		  → param2 (선택)  : [number] function이 적용된 행 인덱스
		  → param3 (선택)  : [number] function이 적용된 열 인덱스
		  → param4 (선택)  : [string] 셀 데이터
		  → param5 (선택)  : [object] json object 형식의 행 데이터
Example 
			
	var datagrid;
	var SBGridProperties = {};
	
	function createGrid(){
		
		SBGridProperties.parentid = 'SBGridArea';
		SBGridProperties.id = 'datagrid';
		SBGridProperties.jsonref = 'ct_data.gridData';
		SBGridProperties.selectmode = 'free';
		SBGridProperties.columns = [
			{caption : ['기본정보','이름'],		ref : 'name',		width : '300px',	style : 'text-align:center',	type : 'output', fixedRenderer : fnRenderer},
			{caption : ['기본정보','나이'],		ref : 'age',		width : '300px',	style : 'text-align:center',	type : 'output' , 
				renderer:function(objGrid, nRow, nCol, strValue, objRowData) {
					return '<input type="button" value="'+strValue+'" onclick="fnRendererClick(' + strValue + ')">';
				}
			}
		];
		datagrid = _SBGrid.create(SBGridProperties);
	};
	
	function fnRendererClick(){
		alert(strValue + ' 버튼클릭');
	}
	
	function fnRenderer(objGrid, nRow, nCol, strValue, objRowData) {
		return '<input type="button" value="'+strValue+'" onclick="fnRendererClick(' + strValue + ')">';
	}
