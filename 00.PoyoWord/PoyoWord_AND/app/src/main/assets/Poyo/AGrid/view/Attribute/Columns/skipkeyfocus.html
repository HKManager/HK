skipkeyfocus

Description

  • ● 키보드로 cell 이동시 지정한 컬럼을 건너뛰도록 설정합니다.

Value

			
	 * 컬럼속성 : skipkeyfocus : value1
	   - value1 (필수) : [boolean] 컬럼의 건너뜀 여부
		  → true			: 컬럼 건너뜀
		  → false (default)	: 컬럼 건너뛰지 않음
