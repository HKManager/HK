input

Description

  • ● 컬럼 type 중 input type을 설정합니다.
  • ● 데이터 입력이 가능한 유형입니다.

Value

			
	 * typeinfo : {maxlength : value1, autonext : value2,  oneclickedit : value3, mask : {alias: value4, unmaskvalue : value5}}
	   - value1 (선택) : [number] 입력 가능한 최대 글자수
	   - value2 (선택) : [boolean] maxlength만큼 입력이 되면 자동으로 다음 셀로 포커스 이동 여부
		  → true			: 포커스 이동
		  → false (default)	: 포커스 이동하지 않음
	   - value3 (선택) : [boolean] 마우스 한 번의 클릭으로 편집 모드 활성화 여부
		  → true			: 마우스 한 번 클릭으로 편집
		  → false (default)	: 마우스 더블 클릭으로 편집
	   - value4 (선택) : [number] 편집 모드에서 데이터 표시 형식
	   - value5 (선택) : [boolean] 실제 데이터를 alias를 적용하지 않은 입력 데이터로 변경 여부
		  → true (default)	: alias를 적용하지 않은 실제 입력한 데이터로 변경
		  → false			: alias를 적용한 데이터로 변경

			{caption : ['기본정보','생년월일'],	ref : 'birth',		width : '90px',	style : 'text-align:center',	type : 'input',			typeinfo : {oneclickedit : true, mask : {alias : '####/##.##', unmaskvalue : true }}},
			{caption : ['기본정보','입사년차'],	ref : 'period',		width : '80px',	style : 'text-align:center',	type : 'output'},

			{caption : ['기본정보','도시'],		ref : 'nation2',	width : '120px',	style : 'text-align:center',	type : 'inputcombo' , 	typeinfo : {ref:'combofilteringData', label:'label', value:'value' , itemcount : 4} },
