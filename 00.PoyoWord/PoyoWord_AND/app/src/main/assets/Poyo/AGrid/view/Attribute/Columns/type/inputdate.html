inputdate

Description

  • ● 컬럼 type 중 inputdate type을 설정합니다.
  • ● 날짜를 입력받거나 캘린더로 선택할 수 있는 복합 유형입니다.

Value

 
			
	 * typeinfo : {dateformat : value1, oneclickedit : value2, locale : value3, clearbutton : value4, multiple : value5, position : value6, alias : value7, calendartype : value8, yearrange : value9}
	   - value1 (필수) : [string] 달력에 적용할 원본 데이터 형식
		  → 'yymmdd' | 'yy.mm.dd' | 'yy-mm-dd' | 'yy/mm/dd'	: 원본 데이터 형식이 연월일 순서
		  → 'mmddyy' | 'mm.dd.yy' | 'mm-dd-yy' | 'mm/dd/yy'	: 원본 데이터 형식이 월일년 순서
		  → 'ddmmyy' | 'dd.mm.yy' | 'dd-mm-yy' | 'dd/mm/yy'	: 원본 데이터 형식이 일월년 순서
	   - value2 (선택) : [boolean] 셀에서 마우스 한번 클릭으로 달력 및 편집 모드 활성화 여부
		  → true			: 마우스 한 번 클릭으로 달력 활성화
		  → false (default)		: 마우스 더블 클릭으로 달력 활성화
	   - value3 (선택) : [string] 달력의 기본 언어
		  → 'ko' (default)		: 한국어
		  → 'en'			: 영어
	   - value4 (선택) : [boolean] 달력에 오늘/지움 버튼 표시 여부
		  → true			: 오늘/지움 버튼 표시
		  → false (default)		: 오늘/지움 버튼 미표시
	   - value5 (선택) : [number] 달력의 표시 개수 (default : 1)
	   - value6 (선택) : [string] 달력의 표시 위치
		  → 'bottom' (default)	: 셀의 아래쪽에 달력 아이콘을 표시
		  → 'top'		        : 셀의 위쪽에 달력 아이콘을 표시
		  → 'right'		        : 셀의 오른쪽에 달력 아이콘을 표시
		  → 'left'		        : 셀의 왼쪽에 달력 아이콘을 표시
	   - value7 (선택) : [string] 셀의 편집 모드에서 데이터 표시 형식 (default : yyyy-mm-dd')
	   - value8 (선택) : [string] inputdate, datepicker 컬럼 타입 달력 종류 선택 하는 속성
		  → 'date'		        : 년도, 월, 일 선택
		  → 'year'		        : 년도 선택
		  → 'month'		        : 월 선택
		  → 'yearmonth'	        : 년도, 월 선택
	   - value9 (선택) : [number] 해당 년도 기준 선택 상자 표시 범위 (default : 10)
