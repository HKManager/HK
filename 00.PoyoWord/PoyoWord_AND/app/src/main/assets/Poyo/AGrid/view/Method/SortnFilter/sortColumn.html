sortColumn

Description

  • ● 특정 열을 오름차순/내림차순으로 정렬합니다.

Parameter

	
	 * 메소드 : datagrid.sortColumn(param1, param2);
	   - param1 (필수) : [number] 열 인덱스
	   - param2 (필수) : [string] 정렬 방법
		  → 'asc'		: 오름차순
		  → 'desc'		: 내림차순

	function sortColumn1(){
