setCellData

Description

  • ● 선택한 셀에 데이터를 입력하거나 변경합니다.

Parameter

			
	 * 메소드 : datagrid.setCellData(param1, param2, param3, param4, param5);
	   - param1 (필수) : [number] 행 인덱스  
	   - param2 (필수) : [number] 열 인덱스
	   - param3 (필수) : [string] 입력 데이터 값
	   - param4 (선택) : [boolean] 데이터 갱신 후 그리드의 refresh 여부
		  → true (default)	: refresh 실행 
		  → false			: refresh 미실행
	   - param5 (선택) : [boolean] 데이터 갱신 후 행 상태 정보 변경 여부
		  → true			: 행 상태 정보 update로 변경
		  → false (default)	: 행 상태 정보 update로 변경하지 않음
