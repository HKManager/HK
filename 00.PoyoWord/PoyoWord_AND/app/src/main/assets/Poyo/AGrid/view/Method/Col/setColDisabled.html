setColDisabled

Description

  • ● 지정한 열의 disabled 상태를 설정합니다.

Parameter

	
	 * 메소드 : datagrid.setColDisabled(param1, param2, param3);
	   - param1 (필수)  : [number] 열 인덱스
	   - param2 (필수)  : [boolean] disabled 여부
		  → true			: disabled 설정
		  → false			: disabled 해제 
	   - param3 (선택)  : [boolean] disabled 상태 변경 후 그리드의 refresh 여부
		  → true			: 메소드 작동 후 바로 refresh
		  → false (default)	: 메소드 작동 후 refresh 하지않음

		var value = datagrid.getColDisabled(nCol);

	function fnMethod(){
