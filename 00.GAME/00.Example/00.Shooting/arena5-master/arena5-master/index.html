Chrome/FireFox Gamepad API support [+]
1. Plug in XBox360/PS3 USB Gamepad.
2. Refresh Arena5 page - press 'A'/'X' button on pad to start.
3. Use the analogue sticks to move and fire.

OTHER KEYS

T - Toggle sound (Chrome/FireFox only)

[ ] - Zoom in/out world

ESC - Pause game
